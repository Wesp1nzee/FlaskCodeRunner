# FlaskCodeRunner

FlaskCodeRunner — это веб-приложение, позволяющее выполнять Python-код на сервере в режиме реального времени. Этот проект представляет собой простую онлайн-IDE, где пользователи могут писать, запускать и получать результаты выполнения кода прямо в браузере.

## Особенности проекта

- **Веб-интерфейс IDE**: Удобный редактор для ввода Python-кода с автодополнением и подсветкой синтаксиса.
- **Поддержка ввода/вывода**: Реализована передача пользовательского ввода в программу и отображение результатов выполнения в режиме реального времени.
- **Механизм отмены выполнения**: Пользователь может отменить выполнение кода даже в случае бесконечных циклов.
- **Безопасность**: Проверка и фильтрация запрещенных операций для предотвращения выполнения опасного кода.

## Структура репозитория

```
.
├── app.py             # Главный файл приложения на Flask; маршруты и обработчики SocketIO.
├── code_executor.py   # Логика выполнения Python-кода в отдельном процессе с потоками для ввода/вывода.
├── extensions.py      # Инициализация расширений, например SocketIO.
├── requirements.txt   # Список зависимостей проекта.
├── static/            # Статические файлы (JS, CSS и др.).
└── templates/         # HTML-шаблоны для отображения страниц веб-приложения.
```

## Установка и запуск

1. **Клонируйте репозиторий:**

   ```bash
   git clone https://github.com/Wesp1nzee/FlaskCodeRunner.git
   ```

2. **Перейдите в директорию проекта и создайте виртуальное окружение:**

   ```bash
   cd FlaskCodeRunner
   python3 -m venv .venv
   ```

3. **Активируйте виртуальное окружение и установите зависимости:**

   ```bash
   source .venv/bin/activate
   pip install -r requirements.txt
   ```

4. **Запустите приложение:**

   ```bash
   python app.py
   ```

5. **Откройте браузер и перейдите по адресу:**

   ```
   http://localhost:5000
   ```

## Использование

- Введите ваш Python-код в редактор на главной странице.
- Нажмите кнопку **Run Code** для запуска кода.
- Результаты выполнения будут отображаться в окне вывода в режиме реального времени.
- Если выполнение кода необходимо отменить, используйте кнопку **Cancel Execution**.
